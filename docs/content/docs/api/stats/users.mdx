## Get Users

<div className="grid grid-cols-1 xl:grid-cols-2 gap-8">
<div>

```
GET /api/users/:site
```

Returns a paginated list of users with their analytics data. Supports filtering by identified users only.

### Path Parameters

<TypeTable
  type={{
    site: {
      description: 'Site ID or site identifier',
      type: 'string | number',
      required: true,
    },
  }}
/>

### Query Parameters

Accepts all [Common Parameters](/docs/api/stats/reference#common-parameters) plus the following:

<TypeTable
  type={{
    page: {
      description: 'Page number (1-indexed)',
      type: 'number',
      default: 1,
    },
    page_size: {
      description: 'Number of results per page',
      type: 'number',
      default: 20,
    },
    sort_by: {
      description: 'Field to sort by',
      type: 'string',
      typeDescription: '"first_seen" | "last_seen" | "pageviews" | "sessions" | "events"',
      default: 'last_seen',
    },
    sort_order: {
      description: 'Sort direction',
      type: 'string',
      typeDescription: '"asc" | "desc"',
      default: 'desc',
    },
    identified_only: {
      description: 'Only return users with a custom identified_user_id',
      type: 'string',
      typeDescription: '"true" | "false"',
      default: 'false',
    },
  }}
/>

### Response

<TypeTable
  type={{
    data: {
      description: 'Array of user objects',
      type: 'User[]',
    },
    totalCount: {
      description: 'Total number of users across all pages',
      type: 'number',
    },
    page: {
      description: 'Current page number',
      type: 'number',
    },
    pageSize: {
      description: 'Number of results per page',
      type: 'number',
    },
  }}
/>

#### User Object

<TypeTable
  type={{
    user_id: {
      description: 'Device fingerprint identifier',
      type: 'string',
    },
    identified_user_id: {
      description: 'Custom user ID when identified, empty string otherwise',
      type: 'string',
    },
    traits: {
      description: 'Custom user traits/properties',
      type: 'object | null',
    },
    country: {
      description: 'Country code',
      type: 'string',
    },
    region: {
      description: 'Region/state',
      type: 'string',
    },
    city: {
      description: 'City name',
      type: 'string',
    },
    language: {
      description: 'Browser language',
      type: 'string',
    },
    browser: {
      description: 'Browser name',
      type: 'string',
    },
    operating_system: {
      description: 'Operating system name',
      type: 'string',
    },
    device_type: {
      description: 'Device type (desktop, mobile, tablet)',
      type: 'string',
    },
    pageviews: {
      description: 'Total pageviews by this user',
      type: 'number',
    },
    events: {
      description: 'Total custom events by this user',
      type: 'number',
    },
    sessions: {
      description: 'Total sessions by this user',
      type: 'number',
    },
    hostname: {
      description: 'First hostname visited',
      type: 'string',
    },
    first_seen: {
      description: 'First activity timestamp',
      type: 'string',
      typeDescription: 'ISO 8601 datetime',
    },
    last_seen: {
      description: 'Most recent activity timestamp',
      type: 'string',
      typeDescription: 'ISO 8601 datetime',
    },
  }}
/>
