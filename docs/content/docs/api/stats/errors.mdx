## Get Error Names

<div className="grid grid-cols-1 xl:grid-cols-2 gap-8">
<div>

```
GET /api/error-names/:site
```

Returns a list of unique error messages with occurrence counts and affected session counts.

### Path Parameters

<TypeTable
  type={{
    site: {
      description: 'Site ID or site identifier',
      type: 'string | number',
      required: true,
    },
  }}
/>

### Query Parameters

Accepts all [Common Parameters](/docs/api/getting-started#common-parameters) plus the following:

<TypeTable
  type={{
    page: {
      description: 'Page number (1-indexed). Enables paginated response format.',
      type: 'number',
      required: false,
    },
    limit: {
      description: 'Number of results per page',
      type: 'number',
      default: 10,
    },
  }}
/>

### Response

<TypeTable
  type={{
    data: {
      description: 'Array of error objects (or paginated wrapper)',
      type: 'Error[] | PaginatedResponse',
    },
  }}
/>

#### Error Object

<TypeTable
  type={{
    value: {
      description: 'Error message',
      type: 'string',
    },
    errorName: {
      description: 'Error type (TypeError, ReferenceError, etc.)',
      type: 'string',
    },
    count: {
      description: 'Total occurrences',
      type: 'number',
    },
    sessionCount: {
      description: 'Unique sessions affected',
      type: 'number',
    },
    percentage: {
      description: 'Percentage of total error sessions',
      type: 'number',
    },
  }}
/>

## Get Error Events

<div className="grid grid-cols-1 xl:grid-cols-2 gap-8">
<div>

```
GET /api/error-events/:site
```

Returns individual error occurrences for a specific error message, with full context including stack traces.

### Path Parameters

<TypeTable
  type={{
    site: {
      description: 'Site ID or site identifier',
      type: 'string | number',
      required: true,
    },
  }}
/>

### Query Parameters

Accepts all [Common Parameters](/docs/api/getting-started#common-parameters) plus the following:

<TypeTable
  type={{
    errorMessage: {
      description: 'The error message to filter by',
      type: 'string',
      required: true,
    },
    page: {
      description: 'Page number (1-indexed)',
      type: 'number',
      required: false,
    },
    limit: {
      description: 'Number of results per page',
      type: 'number',
      default: 20,
    },
  }}
/>

### Response

<TypeTable
  type={{
    data: {
      description: 'Array of error event objects',
      type: 'ErrorEvent[]',
    },
  }}
/>

#### ErrorEvent Object

    browser_version: {
      description: 'Browser version',
      type: 'string',
    },
    operating_system: {
      description: 'Operating system',
      type: 'string',
    },
    device_type: {
      description: 'Device type',
      type: 'string',
    },
    country: {
      description: 'Country code',
      type: 'string',
    },
    message: {
      description: 'Error message',
      type: 'string',
    },
    stack: {
      description: 'Stack trace',
      type: 'string | null',
    },
    fileName: {
      description: 'Source file name',
      type: 'string | null',
    },
    lineNumber: {
      description: 'Line number in source',
      type: 'number | null',
    },
    columnNumber: {
      description: 'Column number in source',
      type: 'number | null',
    },
  }}
/>

## Get Error Time Series

<div className="grid grid-cols-1 xl:grid-cols-2 gap-8">
<div>

```
GET /api/error-bucketed/:site
```

Returns error occurrence counts over time for a specific error message. Useful for tracking error trends and identifying when issues started.

### Path Parameters

<TypeTable
  type={{
    site: {
      description: 'Site ID or site identifier',
      type: 'string | number',
      required: true,
    },
  }}
/>

### Query Parameters

Accepts all [Common Parameters](/docs/api/getting-started#common-parameters) plus the following:

<TypeTable
  type={{
    errorMessage: {
      description: 'The error message to filter by',
      type: 'string',
      required: true,
    },
    bucket: {
      description: 'Time bucket size for grouping data',
      type: 'TimeBucket',
      typeDescription: '"minute" | "five_minutes" | "hour" | "day" | "week" | "month"',
      required: true,
    },
  }}
/>

### Response

<TypeTable
  type={{
    success: {
      description: 'Whether the request succeeded',
      type: 'boolean',
    },
    data: {
      description: 'Array of time-bucketed counts',
      type: 'ErrorBucket[]',
    },
  }}
/>

#### ErrorBucket Object

<TypeTable
  type={{
    time: {
      description: 'Start of the time bucket',
      type: 'string',
      typeDescription: 'ISO 8601 datetime',
    },
    error_count: {
      description: 'Number of errors in this bucket',
      type: 'number',
    },
  }}
/>

</div>
