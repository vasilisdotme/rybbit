## Get Events

<div className="grid grid-cols-1 xl:grid-cols-2 gap-8">
<div>

```
GET /api/events/:site
```

Returns a paginated list of events (pageviews, custom events, and outbound clicks).

### Path Parameters

<TypeTable
  type={{
    site: {
      description: 'Site ID or site identifier',
      type: 'string | number',
      required: true,
    },
  }}
/>

### Query Parameters

Accepts all [Common Parameters](/docs/api/getting-started#common-parameters) plus the following:

<TypeTable
  type={{
    page: {
      description: 'Page number (1-indexed)',
      type: 'number',
      default: 1,
    },
    page_size: {
      description: 'Number of results per page',
      type: 'number',
      default: 20,
    },
  }}
/>

### Response

<TypeTable
  type={{
    data: {
      description: 'Array of event objects',
      type: 'Event[]',
    },
    pagination: {
      description: 'Pagination metadata',
      type: 'Pagination',
    },
  }}
/>

#### Event Object

<TypeTable
  type={{
    timestamp: {
      description: 'Event timestamp',
      type: 'string',
      typeDescription: 'ISO 8601 datetime',
    },
    type: {
      description: 'Event type',
      type: 'string',
      typeDescription: '"pageview" | "custom_event" | "outbound"',
    },
    event_name: {
      description: 'Custom event name',
      type: 'string',
    },
    properties: {
      description: 'Event properties as JSON string',
      type: 'string',
    },
    user_id: {
      description: 'User identifier',
      type: 'string',
    },
    pathname: {
      description: 'Page path',
      type: 'string',
    },
    hostname: {
      description: 'Hostname',
      type: 'string',
    },
    page_title: {
      description: 'Page title',
      type: 'string',
    },
    referrer: {
      description: 'Referrer URL',
      type: 'string',
    },
    browser: {
      description: 'Browser name',
      type: 'string',
    },
    operating_system: {
      description: 'Operating system',
      type: 'string',
    },
    country: {
      description: 'Country code',
      type: 'string',
    },
    device_type: {
      description: 'Device type',
      type: 'string',
    },
  }}
/>

</div>
